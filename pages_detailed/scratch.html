О курсе

Курс по Scratch создан специально для детей 7–9 лет, которые делают первые шаги в мире программирования. Мы учим через игру и творчество: дети создают собственные мультфильмы, анимации и простые игры с помощью визуального языка Scratch.

Программа построена так, чтобы ребенок без стресса и с интересом погружался в основы логики, алгоритмов и креативного мышления. На каждом занятии — новый проект, который можно показать родителям и друзьям!

Сертификат

По окончании курса выдается красочный сертификат о прохождении

Формат обучения

Онлайн-занятия

  • Занятия проходят на нашей платформе с интерактивной доской
  • Можно заниматься из любой точки мира
  • Записи занятий доступны в личном кабинете

Групповые занятия

  • Группы до 8 человек для индивидуального подхода
  • 2 занятия в неделю по 60 минут
  • Домашние задания с проверкой преподавателем

Технические требования

  • Компьютер/ноутбук с Windows, macOS или Linux
  • Стабильный интернет для онлайн-занятий
  • Наушники с микрофоном для общения с преподавателем
  • Установленный Scratch (или доступ к онлайн-версии) (поможем с установкой на первом занятии)
  • Возраст: 7–9 лет. Предыдущий опыт программирования не требуется!
